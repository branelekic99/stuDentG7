# README #

Root repository for StuDent project with frontend and api

### How do I get set up? ###

* Make sure you have Node v14.15.1+ installed

### How to run the API? ###

* Navigate to api directory
* Run ```npm install```
* Run ```npm start``` 

### How to run Client? ###

* Navigate to client directory
* Run npm install
* Run npm start
